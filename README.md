\# 🚗 GolpaMotor Warranty Management System



A multi-layer ASP.NET Core MVC web application for managing automotive products and warranty cards. The project provides a modern administrative panel with product management, warranty management, Excel import, and dashboard capabilities.



> \*\*Status:\*\* 🚧 Under Active Development



\---



\## 📖 Overview



GolpaMotor is an administrative system developed for managing automotive spare parts and warranty cards.



The application is built using ASP.NET Core MVC with a layered architecture to provide a clean, maintainable, and scalable solution.



The project is continuously being improved by adding new modules and enhancing existing features.



\---



\# ✨ Features



\### 📦 Product Management



\- Create, edit and delete products

\- Product image upload

\- Product availability management

\- Product scoring

\- Product search

\- Pagination



\### 🛡 Warranty Management



\- Create warranty cards

\- Edit warranty information

\- Delete warranty cards

\- Warranty status management

\- Search \& filtering



\### 📥 Excel Import



\- Import warranty cards from Excel

\- Duplicate validation

\- Import summary report

\- EPPlus integration



\### 📊 Dashboard



\- Product statistics

\- Warranty statistics

\- Summary cards

\- Responsive admin dashboard



\### 🔐 Authentication



\- ASP.NET Core Identity

\- Login system

\- Role-based authorization



\### 🎨 User Interface



\- Responsive design

\- Bootstrap 5

\- RTL support

\- SweetAlert2

\- Modern admin panel
